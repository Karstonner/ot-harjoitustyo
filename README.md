## Käyttöohje
1. Asenna poetry-riippuvuudet juurihakemistossa:
```bash
poetry install
```
2. Siirry virtuaaliympäristöön:
```bash
poetry shell
```
3. Käynnistä sovellus:
```bash
poetry run invoke start
```
